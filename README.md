# PayStore (PhonePe Clone)

PayStore is a responsive, modern web application designed as a clone of the PhonePe interface. It serves as a unified platform for payments, utility bill recharges, and everyday shopping. Built with pure HTML, CSS, and JavaScript, it features a clean layout, interactive elements, and a smooth user experience.

## Features

- **Responsive Design**: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices.
- **Hero Slider**: An automatic image slider highlighting top offers and services.
- **Quick Transfers & Services**: A dedicated section to quickly navigate through payment options and regular services like mobile recharge, DTH, and electricity.
- **Built-in Store**: An integrated e-commerce section displaying products with a "View All" toggle to expand the product list.
- **Interactive UI Elements**: Hover effects, smooth transitions, and mobile-friendly navigation menus.
- **Modern Aesthetics**: Utilizes CSS Grid/Flexbox, dynamic shadows, and a clean purple-themed color palette reminiscent of modern fintech apps.

## Technologies Used

- **HTML5**: For semantic structuring of the application.
- **CSS3**: For styling, including Flexbox, Grid layouts, and responsive media queries.
- **JavaScript (Vanilla)**: For handling interactive elements like the image slider, mobile menu toggle, and "View All" functionality.
- **Ionicons**: For lightweight, sharp vector icons.
- **Google Fonts**: Uses 'Inter' for clean and readable typography.

## Getting Started

To run this project locally, simply follow these steps:

1. Clone this repository to your local machine:
   ```bash
   git clone <your-repo-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd frontpro # or the directory name you cloned into
   ```
3. Open the `index.html` file in your preferred web browser:
   - You can simply double-click the `index.html` file in your file explorer.
   - Alternatively, use an extension like [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code for a better development experience with live reloading.
